# 📊 End to End Sales Analysis
## 🧠 Overview
This project presents an end-to-end **Sales Analysis Dashboard** built using **Python, MySQL, and Power BI**.

It demonstrates the complete data workflow:
- Data cleaning and preprocessing using Python  
- Data analysis using MySQL  
- Interactive dashboard creation using Power BI  

The dashboard analyzes business performance across **sales, products, regions, and customers**.

---

## 🎯 Objectives
- Clean and prepare raw data for analysis  
- Analyze sales trends and performance  
- Identify top and underperforming products  
- Evaluate regional sales distribution  
- Understand customer behavior and value  

---

## 🛠️ Tools & Technologies
- **Python (Pandas, NumPy)** → Data Cleaning  
- **MySQL** → Data Analysis  
- **Power BI** → Data Visualization  
- **Kaggle** → Dataset Source  

---

## 📂 Project Workflow

### 1️⃣ Data Collection
- Dataset sourced from Kaggle  
- Contains sales transactions, customer, product, and regional data  

---

### 2️⃣ Data Cleaning (Python)
- Loaded dataset using Pandas  
- Checked data types and structure  
- Handled missing values  
- Removed duplicate records  
- Prepared clean dataset for analysis  

---

### 3️⃣ Data Analysis (MySQL)
- Performed aggregations:
  - Total Sales  
  - Total Orders  
  - Customer Count  
- Analyzed:
  - Top products  
  - Regional performance  
  - Customer purchase behavior  

---

### 4️⃣ Data Visualization (Power BI)

#### 🟢 Executive Overview
- Total Sales, Orders, Customers, Avg Order Value  
- Sales trend over time  
- Sales distribution by segment  

#### 🔵 Product Analysis
- Top 10 Products by Sales  
- Bottom 10 Products by Sales  
- Sales by Category/Sub-category  

#### 🟣 Regional Analysis
- Sales by Region and State  
- Regional trends  
- Orders distribution  

#### 🟡 Customer Analysis
- Customer Lifetime Value (CLV)  
- Top Customers  
- Avg Spend per Customer  
- Repeat Customer % by State  
- Customer segmentation
