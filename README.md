📊 End to End Sales Analysis
🧠 Overview

This project presents an end-to-end Sales Analysis Dashboard built using Python, MySQL, and Power BI.

It demonstrates the complete data workflow:

Data cleaning and preprocessing using Python
Data analysis using SQL (MySQL)
Interactive dashboard development using Power BI

The dashboard provides insights into sales performance, product trends, regional distribution, and customer behavior.

🎯 Objectives
Clean and preprocess raw sales data
Perform exploratory data analysis
Analyze business performance across multiple dimensions
Identify key trends in products, regions, and customers
Build an interactive dashboard for decision-making
🛠️ Tools & Technologies
Python (Pandas) → Data Cleaning & Exploration
MySQL → Data Analysis & Querying
Power BI → Data Visualization
Kaggle → Dataset Source
📂 Project Workflow
1️⃣ Data Collection
Dataset sourced from Kaggle
Contains sales transactions, product details, customer data, and regional information
2️⃣ Data Cleaning (Python)
Loaded dataset using Pandas
Checked data types and structure
Handled missing values
Removed duplicate records
Prepared clean dataset for analysis
3️⃣ Data Analysis (MySQL)
Performed SQL queries for:
Total Sales, Orders, and Customers
Product performance analysis
Regional sales distribution
Customer purchase behavior
Created structured outputs for dashboard use
4️⃣ Data Visualization (Power BI)
🟢 Executive Overview
Total Sales, Orders, Customers, Avg Order Value
Sales trend over time
Sales distribution by segment
🔵 Product Analysis
Top 10 Products by Sales
Bottom 10 Products by Sales
Sales by Category and Sub-category
🟣 Regional Analysis
Sales by Region and State
Regional sales trends
Order distribution across regions
🟡 Customer Analysis
Customer Lifetime Value (CLV)
Top Customers
Average Spend per Customer
Repeat Customer % by State
Customer segmentation by category
📊 Key Insights
Revenue is concentrated among a small group of top-performing products
Sales distribution varies significantly across regions
A small segment of customers contributes a large portion of total revenue
Customer loyalty differs across geographic regions
